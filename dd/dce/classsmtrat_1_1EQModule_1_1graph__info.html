Detailed Description

template<typename Settings>
class smtrat::EQModule< Settings >::graph_info

We store a graph_info element for every term (function instance or variable) that we encounter.

Every graph_info element contains an index into the union-find datastructures, a list of explicit, implicit and inequality edges.
