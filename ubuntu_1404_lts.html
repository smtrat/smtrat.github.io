Ubuntu 14.04 LTS 64Bit

This is an example how the installation on Ubuntu 14.04 LTS 64Bit looks like. We assume that you start with a fresh installation and know how to install software packages.

  • Install the following packages:
    • git
    • cmake and cmake-curses-gui
    • libcln6, libcln-dev and libgmp-dev
    • libeigen3-dev
    • g++ or clang
    • libboost-dev
    • doxygen
  • Checkout CArL:
    1 git clone https://<user>@srv-i2.informatik.rwth-aachen.de:8443/git/carl.git
  • Create build folder:
    1 cd carl/
    2 mkdir build
    3 cd build/
  • Run CMake:
    1 cmake ../
    By default, c++ will be used as compiler which points to g++. This, and many other options, can be configured using ccmake.
  • Build libcarl:
    1 make lib_carl
    This builds the shared library build/libcarl.so.
  • Build and run tests:
    1 make googletest
    2 make
    3 make test
    The individual tests are stored in build/bin/.
  • Build documentation:
    1 make doc
    The documentation is stored in build/doc/.
