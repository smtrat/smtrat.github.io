Detailed Description

template<typename T>
class benchmax::RandomizationAdaptor< T >

Provides iteration over a given std::vector in a pseudo-randomized order. Internally, it computes a factor that is coprime to the number of jobs (and thus generates all indices). We choose the factor in the order of the square root of the number of jobs.
