Detailed Description

template<typename Coeff>
class carl::cad::PolynomialOwner< Coeff >

This class manages the ownership of pointers to UnivariatePolynomial objects.

It is intended to be subclassed, for example by the CAD class.

The ownership of a pointer can be delegated by calling takeOwnership(). A pointer that has been delegated must not be deleted manually. This will automatically be done when this class is destroyed.
