Detailed Description

Represent a dynamic root, also known as an "indexed root expression".

It's a generalization of an algebraic real(= univariate poly having roots + root index). It uses a multivariate poly(with distinguished main/substitution variable) and an root index. If we want to "evaluate" this root, we need to plug in a subpoint, i.e., algebraic-real values for all non-main variables. The result is basically an algebraic real. However, since this algebraic real depends on the subpoint, you can also think of this as a multi-argument function f: algReal^k -> algReal.
