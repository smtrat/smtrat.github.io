Detailed Description

template<typename Rational, typename Poly>
class carl::Model< Rational, Poly >

Represent a collection of assignments/mappings from variables to values.

We use a ModelVariable to abstract over the different kinds of variables in CARL, and a ModelValue to abstract over the different kinds of values for these variables. Most notably, a value can be a "carl::ModelSubstitution" whose value depends on the values of other variables in the Model. If such other variables are erased from the model, this substitution is evaluated and replaced by the result.
