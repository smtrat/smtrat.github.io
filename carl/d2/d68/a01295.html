Evaluate the given constraint 'c' at the given 'point' based on the variable order given by 'variables'.

If a variable is assigned a numeric representation, the corresponding value is directly plugged in. All assignments of interval representations are passed on to evaluate(Constraint, RANIRMap). Note that the number of variables must match the dimension of the 'point', all variables of 'c' must appear in 'variables' and that 'variables' must not mention any additional variables.
