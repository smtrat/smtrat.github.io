template<typename Pol>
unsigned carl::Constraint< Pol >::evaluate ( const EvaluationMap< Interval< typename carl::UnderlyingNumberType< Pol >::type >> &  _assignment) const

Checks whether the given interval assignment may fulfill the constraint.

Note that the assignment must be complete. There are three possible outcomes:

  • True (4), i.e. all actual assignments satisfy the constraint: $p ~_ 0 True$
  • Maybe (3), i.e. some actual assignments satisfy the constraint: $p ~_ 0 ?$
  • Not null (2), i.e. all assignments that make the constraint evaluate not to zero satisfy the constraint: $p ~_ 0 p 0$
  • Null (1), i.e. only assignments that make the constraint evaluate to zero satisfy the constraint: $p ~_ 0 p_ = 0$
  • False (0), i.e. no actual assignment satisfies the constraint: $p ~_ 0 False$
    Parameters
    _assignmentVariable assignment.
    Returns
    0, 1 or 2.
