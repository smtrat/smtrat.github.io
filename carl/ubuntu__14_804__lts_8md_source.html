1 Ubuntu 14.04 LTS 64Bit {#ubuntu_1404_lts}
2 ==========
3 
4 This is an example how the installation on Ubuntu 14.04 LTS 64Bit looks like.
5 We assume that you start with a fresh installation and know how to install software packages.
6 
7 - Install the following packages:
8  - `git`
9  - `cmake` and `cmake-curses-gui`
10  - `libcln6`, `libcln-dev` and `libgmp-dev`
11  - `libeigen3-dev`
12  - `g++` or `clang`
13  - `libboost-dev`
14  - `doxygen`
15 
16 - Checkout CArL:
17 @code
18 git clone https://github.com/smtrat/carl.git
19 @endcode
20 
21 - Create `build` folder:
22 @code
23 cd carl/
24 mkdir build
25 cd build/
26 @endcode
27 
28 - Run CMake:
29 @code
30 cmake ../
31 @endcode
32 By default, `c++` will be used as compiler which points to `g++`. This, and many other options, can be configured using `ccmake`.
33 
34 - Build `libcarl`:
35 @code
36 make lib_carl
37 @endcode
38 This builds the shared library `build/libcarl.so`.
39 
40 - Build and run tests:
41 @code
42 make
43 make test
44 @endcode
45 The individual tests are stored in `build/bin/`.
46 
47 - Build documentation:
48 @code
49 make doc
50 @endcode
51 The documentation is stored in `build/doc/`.
