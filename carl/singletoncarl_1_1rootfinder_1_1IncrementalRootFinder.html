Detailed Description

template<typename Number, typename Comparator>
class carl::rootfinder::IncrementalRootFinder< Number, Comparator >

This class implements an AbstractRootFinder that has an interface to calculate the real roots incrementally.

Using the method next(), the next real root can be obtained. The root finder uses a bisection approach internally and implements multiple strategies how the bisection is performed.
