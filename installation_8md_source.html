13 Quick installation guide
14 --------------------------------------------
15 - Make sure all @subpage dependencies "dependencies" are available.
16 - Download the latest release or clone the git repository from https://github.com/smtrat/smtrat.git
17 - Prepare the build.
18 @code{.sh}
19 $ mkdir build && cd build && cmake ..
20 @endcode
21 - Build SMT-RAT (with tests and documentation).
22 @code{.sh}
23 $ make
24 @endcode
